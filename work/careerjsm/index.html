Helping job-seekers manage their job search

Job seekers have a lot to worry about. Sometimes, they've been out of the job hunt for a while and need to refresh their skills. If they've just lost a job, they can be in a sensitive emotional state. By providing each user a tailored experience that considers all sorts of factors, I've increased both sign-ups and user engagement.

When designing the onboarding/registration workflow, I used thoughtful copy, formatting, and animation, to ease the user into the product with as little friction as possible. The resulting update increased user sign-up by close to 100%.

An iteration of a user flow for signing up.

Meeting user & business needs

To address our user's needs, I conducted surveys and facilitated team design sprints to set new feature requirements.

I've worked on implementing a resume builder, a networking dashboard, a job tracker, a learning center, and an admin dashboard to manage it all (just to name a few).

A blank networking dashboard that a user might see.

Measuring impact

It's not easy to determine how well new features are working, so I make sure to record, gather, and analyze data to evaluate target KPIs. I present findings to the team to then address issues and re-prioritize tasks if necessary.

All pages are mobile compatible and utilizes Progressive Web App technology.

Quality assurance

With my keen sense of detail, I audit and patch problems to make sure the user experience is as smooth as possible. I've taken on responsive design, web accessibility (WCAG 2.0), localization, and bug tracking reviews.
